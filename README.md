# Path Finder and Maze Generation
 Use A* algorithm to find the optimal solution to mazes which have been drawn or procedurally generated using randomised Kruskal's algorithm.
 
 Black -> Walls |
 Green -> Start |
 Red -> Goal |
 Yellow -> Optimal path |
 Purple -> Tested nodes
